# Rustic Rhythms – Music Store Front-End

## Overview
**Rustic Rhythms** is a simulated multi-page online music store website built using **HTML, CSS, XML, XSLT, and DTD**. It showcases albums by genre, supports browsing, and includes forms for login, checkout, and contact. The project focuses entirely on **front-end development**.

---

## Features
- Multi-page navigation: Home, About Us, Store, Contact, Login, Checkout, Thank You.
- Interactive store using **XML + XSLT** to dynamically display album data.
- XML validation with **DTD** to ensure correct structure.
- Consistent styling and layout using **CSS** (flexbox, grid, animations).
- User input forms: Contact, Login, Checkout.
- Responsive design with **Top Bar, Navbar, Sidebar, and Footer** present across pages.

---

## Technologies Used
- **HTML** – Page structure, headings, tables, forms, images.  
- **CSS** – External, internal, and inline styles; flexbox, grid, animations.  
- **XML** – Stores album and store inventory data.  
- **XSLT** – Transforms XML data into XHTML for display.  
- **DTD** – Validates XML structure.

---

## File Structure
- index.html          - Home page
- about.html          - About Us page
- contact.html        - Contact page
- login.html          - User login page
- checkout.html       - Checkout page
- thankyou.html       - Thank You page
- css/styles.css      - External stylesheet
- xml/store.xml       - Store inventory XML
- xml/album.xml       - Individual album XML
- xsl/store.xsl       - XSLT to transform store.xml
- xsl/album.xsl       - XSLT to transform album.xml
- xml/store.dtd       - DTD validation for XML
- images/             - Album covers, logos, and UI images
